## Types
For type variations, the following css classes are built:
- `btn--outline` for outline
- `btn--bare` for bare
- `btn--filled` for filled
- `btn--outline-negative` for outline-negative
- `btn--bare-negative` for bare-negative

## Sizes
For size variations, the following css classes are built:
- `btn--sm` for sm
- `btn--base` for base
- `btn--lg` for lg

## Icons
For icon variations, the following css classes are built:
- `btn__icon-left` for icon left
- `btn__icon-right` for icon right
- `btn__icon-only` for icon only
