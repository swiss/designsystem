# Design system for the Swiss Confederation

## Document for developers

### Github repository

https://github.com/swiss/designsystem

### CSS file

CSS assets can be downloaded on the github repository. Latest version here:

https://github.com/swiss/designsystem/releases

Alognside the css file, you have to include the font files in your project. They are located in the `fonts` folder and can be found here:

https://github.com/swiss/designsystem/tree/main/css/foundations/fonts
