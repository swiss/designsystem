# Spacings

In order to set vertical spacings between generic DOM elements (created in a WYSIWYG editor for example), you can add the class `.vertical-spacing` to the direct parent of these elements.

## Example with vertical spacings

```html
<main id="main-content">
  <section class="section section--default">
    <div class="container vertical-spacing">
      <!-- Content -->
    </div>
  </section>
</main>
```
